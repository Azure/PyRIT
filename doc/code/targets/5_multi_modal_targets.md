# 5. Multi-Modal Targets

Like most of PyRIT, targets can be multi-modal. This notebook collection highlights some common scenarios using multi-modal targets.

Before you begin, ensure you are setup with the correct version of PyRIT installed and have secrets configured as described [here](../../setup/populating_secrets.md).

- [OpenAI Chat Target](./5.1_openai_chat_target.ipynb) (*text + image --> text*)
- [OpenAI Image Target](./5.2_openai_image_target.ipynb) (*text --> image* or *text + image --> image*)
- [OpenAI Video Target](./5.3_openai_video_target.ipynb) (*text --> video*)
- [OpenAI TTS Target](./5.4_openai_tts_target.ipynb) (*text --> audio*)
